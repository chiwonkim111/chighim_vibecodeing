## Cursor AI란 무엇인가요?

Cursor AI는 인공지능이 내장된 코드 편집기입니다. 쉽게 말해, 여러분이 한국어로 "버튼을 만들어줘"라고 말하면 AI가 대신 코드를 작성해주는 마법같은 도구입니다. 2023년 출시 이후 전 세계적으로 폭발적인 인기를 얻고 있으며, 특히 비개발자들 사이에서 큰 주목을 받고 있습니다.

기존의 ChatGPT나 Claude와 다른 점은, Cursor는 **코딩에 특화된 환경**을 제공한다는 것입니다. 파일 관리, 코드 편집, 미리보기까지 모든 것이 하나의 프로그램 안에서 이루어집니다.

## 1단계: Cursor AI 설치하기

Cursor 설치는 매우 간단합니다. 아래 단계를 따라해보세요.

1. 웹 브라우저에서 `cursor.com`에 접속합니다
2. 화면 중앙의 'Download' 버튼을 클릭합니다
3. 본인의 운영체제(Windows 또는 Mac)에 맞는 버전을 선택합니다
4. 다운로드된 파일을 실행하여 설치를 완료합니다
5. 설치가 끝나면 Cursor를 실행하고 계정을 생성합니다

무료 버전으로도 충분히 많은 기능을 사용할 수 있으니, 처음에는 무료 플랜으로 시작하는 것을 추천합니다. Pro 버전은 AI 사용량이 더 많이 필요해질 때 업그레이드하면 됩니다.

## 2단계: 화면 구성 이해하기

Cursor를 처음 실행하면 다소 복잡해 보일 수 있습니다. 하지만 핵심적인 영역만 알면 됩니다.

**왼쪽 사이드바**: 파일과 폴더를 탐색하는 영역입니다. 컴퓨터의 파일 탐색기와 비슷합니다.

**중앙 편집 영역**: 코드를 작성하고 편집하는 메인 공간입니다. 여기서 대부분의 작업이 이루어집니다.

**하단 터미널**: 명령어를 입력하는 영역입니다. 프로젝트를 실행하거나 패키지를 설치할 때 사용합니다.

**오른쪽 AI 채팅**: 가장 중요한 영역입니다! 여기서 AI에게 질문하고, 코드 생성을 요청합니다.

## 3단계: AI와 대화하는 방법

Cursor에서 AI를 호출하는 방법은 여러 가지가 있습니다.

**Ctrl+K (또는 Cmd+K)**: 선택한 코드 영역에 대해 수정이나 생성을 요청할 때 사용합니다. 예를 들어, 빈 파일에서 Ctrl+K를 누르고 "간단한 자기소개 웹페이지를 만들어줘"라고 입력하면 됩니다.

**Ctrl+L (또는 Cmd+L)**: AI 채팅창을 열어 자유롭게 대화합니다. 질문을 하거나, 긴 설명이 필요한 요청을 할 때 유용합니다.

**Ctrl+I (또는 Cmd+I)**: Composer 기능으로, 여러 파일을 한 번에 생성하거나 수정할 때 사용합니다.

## 4단계: 첫 번째 프로젝트 시작하기

이제 실제로 간단한 웹페이지를 만들어봅시다.

1. Cursor에서 새 폴더를 생성합니다 (File > Open Folder > 새 폴더 만들기)
2. 폴더 이름은 "my-first-website"로 지정합니다
3. 왼쪽 사이드바에서 우클릭 > "New File"을 선택하고 `index.html`이라고 이름을 입력합니다
4. 파일이 열리면 Ctrl+K를 누르고 다음과 같이 입력합니다:

```
자기소개 웹페이지를 만들어줘. 
이름은 '홍길동', 직업은 '마케터'로 해줘.
깔끔하고 현대적인 디자인으로 부탁해.
```

5. AI가 코드를 생성하면 "Accept" 버튼을 클릭합니다
6. 파일을 저장하고(Ctrl+S), 해당 파일을 웹 브라우저로 열어봅니다

축하합니다! 여러분의 첫 번째 웹페이지가 완성되었습니다.

## 자주 하는 실수와 해결법

비개발자분들이 처음 Cursor를 사용할 때 자주 겪는 문제들이 있습니다.

**"코드가 이상해요"**: AI가 생성한 코드가 원하는 대로 작동하지 않을 때는, 더 구체적으로 요청해보세요. "버튼을 만들어줘" 대신 "파란색 배경에 흰색 글씨로 '시작하기'라고 쓰인 둥근 버튼을 만들어줘"처럼 상세하게 설명합니다.

**"에러가 발생해요"**: 당황하지 마세요. 에러 메시지를 그대로 복사해서 AI 채팅창에 붙여넣고 "이 에러를 해결해줘"라고 요청하면 됩니다.

**"어디서부터 시작해야 할지 모르겠어요"**: 완성된 결과물을 상상하고, 그것을 AI에게 최대한 자세히 설명하세요. AI는 여러분의 상상을 코드로 번역해주는 통역사입니다.

## 다음 단계로 나아가기

첫 번째 프로젝트를 완성했다면, 이제 더 복잡한 것들에 도전해볼 차례입니다. 개인 포트폴리오 사이트, 간단한 블로그, 랜딩페이지 등 다양한 프로젝트를 시도해보세요. 실패해도 괜찮습니다. AI와 함께라면 언제든 다시 시작할 수 있으니까요.

바이브코딩의 핵심은 완벽한 코드를 작성하는 것이 아니라, **아이디어를 빠르게 현실로 만드는 것**입니다. 여러분도 지금 바로 시작해보세요!
